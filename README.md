# Явление стеклования и плавления

![Демонстрация модели](./media/gif_glass.gif)

Код визуализации находится в `glass_transition_and_melting/script.js`.

## Ключевые моменты

### Основные переменные

- `chains` - массив всех полимерных цепочек.
- `temperature` - текущая температура.
- `isThermoplastic` - тип материала (`true`/`false`).

### Функция `setup()`

`setup()` создает:

1. Холст для рисования.
2. 50 полимерных цепочек.
3. Ползунок для температуры.
4. Кнопку для смены материала.

### Функция `draw()`

Каждый кадр `draw()`:

1. Очищает фон.
2. Обновляет и рисует все цепочки.
3. Показывает информацию.

### Класс `PolymerChain`

- Каждая цепочка имеет положение, длину, цвет и углы сегментов.
- `update()` меняет углы в зависимости от температуры.
- `display()` рисует цепочку на экране.

### Особенности поведения

1. При низкой температуре цепи неподвижны.
2. При средней температуре цепи слегка двигаются.
3. При высокой температуре:
   - Термопласт: цепи сильно двигаются и растягиваются.
   - Реактопласт: цепи не двигаются, что имитирует разложение.

### Для использования

1. Склонируйте этот репозиторий на свой компьютер.
2. Перейдите в папку `glass_transition_and_melting`.
3. Откройте файл `index.html` в браузере.
